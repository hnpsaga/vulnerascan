# VulneraScan

Developer-first vulnerability scanner for modern software projects.

## Status

🚧 Under active development.

This package has been published as an initial name-reservation release.

The purpose of this release is to secure the npm package name while active development continues.

## Planned Features

- Dependency discovery
- Dependency graph generation
- Vulnerability detection
- Vulnerability reporting
- Multi-language support
  - Node.js
  - Maven
  - Gradle
  - Python
- AI-assisted remediation insights
- Local security dashboard

## Installation

```bash
npm install -g @hnpsaga/vulnerascan
```

or

```bash
npx @hnpsaga/vulnerascan --version
```
